¿Qué es HTML?

HTML significa HyperText Markup Language. Es el lenguaje que se utiliza para crear páginas web.

  1. DOCTYPE indica que el documento es HTML5.
  2. HTML contiene toda la página.
  3. HEAD guarda información como el título y estilos.
  4. BODY contiene todo lo visible para el usuario.
  5. Las etiquetas permiten agregar títulos, textos, tablas y formularios.
